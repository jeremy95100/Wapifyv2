Bienvenue sur Compile42

Compile42 transforme un prompt en langage naturel en une application web React complète, fonctionnelle et déployable. Mais avant d'écrire la moindre ligne de code, il co-construit ton projet avec toi — en 8 étapes guidées.

React + TypeScript Tailwind CSS Preview Vite HMR Déploiement 1 clic
Prompt→ Claude AI→ React / TS / Tailwind→ Preview Vite HMR→ Deploy
Ce qui différencie Compile42 de Lovable ou Bolt : il ne génère pas à l'aveugle. Il comprend ton projet — écrans, ton, palette — avant de générer. Résultat : moins d'allers-retours, une app qui ressemble à ce que tu avais en tête.

Pourquoi Compile42

Quatre raisons de construire avec Compile42 plutôt qu'à la main ou avec un générateur classique.

Itération ultra-rapide
Du prompt à la preview live en quelques minutes, avec rechargement à chaud (HMR) à chaque modification.
Co-construction guidée
8 étapes pour cadrer l'idée, le design et les écrans avant la génération — fini les résultats « à côté ».
Tu possèdes ton code
Code React/TS standard, lisible, exportable et poussable sur GitHub. Aucun verrouillage.
Tout est câblé
Auth, base de données et hosting déjà intégrés (Supabase). Déploiement en un clic.

Le parcours en 8 étapes (0 → 7)

C'est le cœur de Compile42. Chaque génération suit ce flux guidé : tu décris, l'IA pose des questions, tu choisis le design, tu valides — puis l'app est générée. Tu peux répondre à chaque étape ou laisser Auto décider.

00
Décris ton idée
Tout commence par un prompt en langage naturel. Décris l'app que tu veux — par ex. « une application pour ma comptabilité en LMNP ». Pas de cahier des charges, pas de jargon technique.
Prompt libreLangage naturel
01
On creuse ensemble
Compile42 pose les vraies questions pour comprendre tes intentions : nom de l'app, type d'espace utilisateur (personnel ou partagé), contraintes métier (ex. régime fiscal), et fonctionnalités prioritaires.
NomUtilisateursContraintesFonctionnalités
02
Choisis ton univers visuel
Sélectionne le ton de l'app parmi 6 ambiances proposées. C'est ce qui donne sa personnalité à l'interface.
Calm minimalEditorial darkPlayful softCorporateWarm artisanTech monospace
03
Valide tes écrans
Compile42 propose la liste des pages adaptées à ton projet (Dashboard, gestion, déclarations…). Tu peux ajouter, renommer ou retirer des écrans avant la génération.
Écrans proposésÉditable
04
Ton hero, ta vision
Trois variations de hero sont générées (par ex. Calm centré, Split avec visuel, Editorial sombre). Tu choisis la direction qui te parle le plus.
3 variantesChoix visuel
05
Palette & typographie
Finalise l'identité visuelle : la palette de couleurs (Emerald calm, Ocean, Warm terra…) et la typographie (DM Sans, Playfair…).
PaletteTypographie
06
Récapitulatif
Une vue d'ensemble avant lancement : nom, contraintes, palette, typo, ton et écrans — ton app résumée en un coup d'œil pour validation finale.
ValidationVue d'ensemble
07
Génération
Compile42 construit ton app React complète, fonctionnelle et déployable — en quelques minutes. Auth, base de données et hosting sont déjà câblés. Modifiable à tout moment après génération.
App ReactDéployable~3 min
À chaque étape, tu peux choisir Auto : Compile42 prend une décision pertinente à ta place pour aller plus vite.

Stack technique

Les apps générées reposent sur une stack moderne et éprouvée.

Frontend

  • React 18 + TypeScript
  • Tailwind CSS
  • Vite 6 — HMR
  • Monaco Editor

Backend & IA

  • Anthropic Claude
  • Express.js
  • BullMQ + Redis
  • esbuild

Infra & données

  • Supabase — DB, Auth, Storage
  • Vercel — frontend
  • Fly.io — build server
  • Upstash — Redis

Exemple — structure d'entrée générée

main.tsx
// Point d'entrée d'une app Compile42
import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App'
import './index.css'

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
)
